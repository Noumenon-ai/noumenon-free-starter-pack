# Install

## Claude Code (30 seconds)

```bash
git clone https://github.com/noumenon-ai/noumenon-free-starter-pack
cp -r noumenon-free-starter-pack/* ~/.claude/skills/
```

## Per-project

```bash
cp -r noumenon-free-starter-pack/* .claude/skills/
```

## Claude.ai

Open any SKILL.md → copy contents → paste into Project Instructions.
